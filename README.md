# AI-Powered QA Automation Engine (MCP Stack)

## 🚀 Overview
This repository serves as a professional showcase for modern **AI-driven Quality Assurance** and **DevOps** methodologies. Built in 2026, it leverages the **Model Context Protocol (MCP)** to bridge the gap between Large Language Models (LLMs) and real-world software testing environments.

As an engineer with a background in **Math, Physics, and Space Tech**, I focus on building high-precision automation tools that go beyond simple script execution, moving towards **Agentic QA systems**.

## 🛠 Tech Stack
- **Languages:** Python (Functional & OOP paradigms)
- **AI Protocols:** Model Context Protocol (MCP), FastMCP SDK
- **Testing:** Pytest, Playwright (Autonomous UI testing)
- **Infrastructure:** GitHub Actions (CI/CD), GitHub Codespaces
- **Knowledge Base:** Integrated with RAG and NotebookLM for architectural decision records.

## 🏗 Project Structure
- `/src`: Custom MCP servers providing high-level tools for AI agents.
- `/tests`: Automated test suites validating both code logic and AI agentic behavior.
- `/.github/workflows`: CI/CD pipelines ensuring 100% infrastructure reliability.

## 🌟 Key Features (Q2 2026)
- **Agentic Testing:** Custom MCP tools that allow LLMs (like Claude or Gemini) to explore, debug, and fix code autonomously.
- **Hybrid Code Style:** Implementation of a robust architecture balancing functional efficiency and object-oriented scalability.
- **Safety First:** Integrated AI Safety checks to prevent command injection and ensure data privacy within agentic workflows.
